CloudPocket Mobile V3

Politique de Confidentialité et Protection des Données

Dernière mise à jour : 21 mai 2026

1. Introduction

CloudPocket Mobile V3 respecte votre vie privée et s'engage à protéger vos données personnelles. Cette Politique de Confidentialité explique comment nous collectons, utilisons, conservons et protégeons vos informations.

2. Données Collectées

2.1 Données Bancaires

CloudPocket collecte les données suivantes exclusivement avec votre consentement explicite :

  • Informations de compte bancaire (IBAN, titulaire)
  • Soldes des comptes bancaires
  • Historique des transactions (24 derniers mois)
  • Données d'authentification bancaire (via Plaid API)

Ces données sont collectées via Plaid, Inc., conforme aux réglementations bancaires européennes (PSD2).

2.2 Données d'Authentification

  • Identifiant unique de l'utilisateur (UUID)
  • Données biométriques (empreinte digitale, reconnaissance faciale)
  • Adresse e-mail
  • Numéro de téléphone

2.3 Données de Paiement

  • Transactions NFC sans contact
  • Transferts P2P (pair-à-pair)
  • Historique des paiements

3. Finalités de Traitement

Nous utilisons vos données pour :

  • Fournir le service : Affichage du solde, transactions, transferts
  • Sécurité : Authentification biométrique, prévention de la fraude
  • Conformité légale : Respect de la PSD2, RGPD, LCB-FT

4. Base Légale

Nous traitons vos données sur la base de :

  • Consentement explicite : Pour l'accès aux données bancaires
  • Exécution du contrat : Pour fournir les services de paiement
  • Obligations légales : Conformité PSD2, anti-blanchiment

5. Partage des Données

Vos données sont partagées UNIQUEMENT pour l'exécution du service :

  • Plaid, Inc. - Agrégateur de données bancaires
  • Prestataires de paiement - Processeurs sécurisés
  • Autorités légales - Si requis par la loi

Nous ne vendons JAMAIS vos données à des tiers commerciaux.

6. Sécurité des Données

CloudPocket implémente les mesures suivantes :

  • Chiffrement AES-256-GCM : Stockage local
  • TLS 1.3+ : Transmissions réseau
  • Authentification biométrique : Accès sécurisé
  • AndroidKeyStore : Stockage des clés
  • Tokens ECDSA : Transferts sécurisés

7. Conservation des Données

Nous conservons vos données selon :

  • Données bancaires : 90 jours après fermeture
  • Historique transactions : 24 mois (PSD2)
  • Données biométriques : Local uniquement
  • Logs sécurité : 13 mois
  • LCB-FT : 5 ans (obligations légales)

8. Droits de l'Utilisateur (RGPD)

Vous disposez des droits suivants :

  • Droit d'accès : Accéder à vos données
  • Droit de rectification : Corriger vos données
  • Droit à l'oubli : Demander la suppression
  • Droit à la portabilité : Exporter vos données
  • Droit d'opposition : S'opposer au traitement

Pour exercer ces droits : privacy@cloudpocket.io

9. Conformité Réglementaire

CloudPocket est conforme à :

  • RGPD (Règlement Général Protection Données)
  • PSD2 (Directive Services de Paiement 2)
  • LCB-FT (Anti-Blanchiment)
  • CNIL (Commission CNIL)
